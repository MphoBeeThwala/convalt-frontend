## How to Run Locally

1. Clone the repository:
   ```sh
   git clone https://github.com/MphoBeeThwala/convalt-frontend
   ```
2. Navigate to the project directory:
   ```sh
   cd convalt-frontend
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
